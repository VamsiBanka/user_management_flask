# User Management System (Python + React + Flask + PostgreSQL)

A full-stack Python web application built using **Flask** and **PostgreSQL** that performs complete **CRUD (Create, Read, Update, Delete)** operations on users.  
This project demonstrates backend development, database integration, and real-world debugging practices.
